# MediGuard AI

*MediGuard AI* is an intelligent medical diagnostic web application that predicts diseases based on *24 clinical blood test parameters. It not only predicts the likely disease but also highlights **key contributing indicators* and flags *abnormal or potentially dangerous values*.

---

## Features

- Predicts diseases such as *Diabetes, Anemia, High Blood Pressure*, and more using a trained ML model.
- Highlights *top 5 contributing features* for each prediction.
- Detects *abnormally high or low values* in blood test parameters.
- Displays *disease description and precautions* for better understanding.
- Interactive *bar chart visualization* of key indicators.
- Easy-to-use web interface.

---

## Technologies Used

- *Backend:* Python, Flask, joblib
- *Frontend:* HTML, JavaScript, Chart.js
- *Machine Learning:* Scikit-learn (Random Forest / Decision Tree)
- *Data Processing:* NumPy
- *Visualization:* Chart.js
